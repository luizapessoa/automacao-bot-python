# Automação de Cadastro com PyAutoGUI

Projeto de automação desenvolvido em Python utilizando PyAutoGUI para automatizar o processo de login e preenchimento de dados em um sistema web.

## Tecnologias utilizadas
- Python
- PyAutoGUI
- Pandas
- Time
- Webbrowser

## Funcionalidades
- Abertura automática do navegador
- Login automático no sistema
- Leitura de base de dados
- Preenchimento automático de formulários
- Automação de tarefas repetitivas

## Objetivo
Automatizar processos manuais de cadastro, aumentando produtividade e reduzindo erros operacionais.
